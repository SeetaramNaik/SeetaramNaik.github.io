Laundry management system

It is a basic PHP project which includes usual database operations like INSERTION, DELETION, UPDATE. This project was done for DBMS mini project. In this website the admin can manage all the data of the laundry customers. And the user may visit to another portal where he can see his laundry status of his clothes and he may pick the clothes if the laundry is done.
